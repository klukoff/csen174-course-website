# CSEN 174: Software Engineering

Course website for CSEN 174 (Software Engineering) at Santa Clara University, Spring 2026.

**Live site:** https://klukoff.github.io/csen174-course-website/

## About

A single-page static HTML site covering the syllabus, weekly schedule, project guide, readings, AI policy, and tools for the course. Hosted via GitHub Pages, deploying from the `main` branch.

## Structure

- `index.html` - the full course website (all tabs in one file)
- `scu-logo-white.png` - SCU logo used in the site header
